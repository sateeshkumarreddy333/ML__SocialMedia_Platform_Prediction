# 📌 Project Overview

This project applies **Machine Learning Classification Algorithms** to predict the **Social Media Platform** based on viral trend information.

The dataset contains various attributes such as **Hashtag**, **Content Type**, **Region**, **Views**, **Likes**, **Shares**, **Comments**, and **Engagement Level**.

The project follows the complete Machine Learning pipeline including:

- 📂 Data Collection
- 🧹 Data Preprocessing
- 🔍 Exploratory Data Analysis
- ⚙ Feature Engineering
- 🤖 Model Training
- 📈 Model Evaluation
- 🏆 Model Comparison

---

# 📊 Dataset

**Dataset Name**

Viral Social Media Trends Dataset

### Features

| Feature | Description |
|----------|-------------|
| Platform | Target Variable |
| Hashtag | Trending hashtag |
| Content Type | Video, Shorts, Post, etc. |
| Region | Country/Region |
| Views | Number of Views |
| Likes | Total Likes |
| Shares | Total Shares |
| Comments | Number of Comments |
| Engagement Level | Low / Medium / High |
